📊 Mutual Fund Analytics Dashboard

A complete Data Analytics project that transforms raw mutual fund data into interactive business insights using Python, SQL, and Power BI.

 🚀 Project Overview

This project focuses on cleaning, analyzing, and visualizing mutual fund data to evaluate industry trends, fund performance, and investment risk through an interactive Power BI dashboard.

🛠️ Tech Stack

* Python (Pandas, NumPy)
* SQL (SQLite)
* Power BI
* Microsoft Excel

 📌 Key Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* SQL Database Integration
* Risk Analysis using **Sharpe Ratio, VaR & CVaR**
* Interactive **3-Page Power BI Dashboard**
* Dynamic Filters & Slicers
* KPI Cards and Trend Analysis

 📊 Dashboard

 Page 1 – Industry Overview

* Industry KPIs
* AUM Trends
* Fund House Analysis

 Page 2 – Fund Performance

* Fund Comparison
* Return Analysis
* Performance Insights

 Page 3 – Risk Analysis

* Sharpe Ratio
* Value at Risk (VaR)
* Conditional Value at Risk (CVaR)
