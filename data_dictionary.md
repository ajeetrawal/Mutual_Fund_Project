# Data Dictionary

## fund_master
Contains master details of all mutual fund schemes.

## nav_history
Contains daily NAV history of each fund.

## aum_by_fund_house
Contains Assets Under Management (AUM) by fund house.

## monthly_sip_inflows
Contains monthly SIP inflows and growth.

## category_inflows
Contains net inflows by category.

## industry_folio_count
Contains folio count across the mutual fund industry.

## scheme_performance
Contains return, expense ratio and performance metrics.

## investor_transactions
Contains investor transaction details.

## portfolio_holdings
Contains stock holdings of mutual fund schemes.

## benchmark_indices
Contains benchmark index closing values.
